**Properties**

 - `/properties/?`

**Listings**

 - `/listings`
> Gets all listings
- `/listings/?`
>Gets single listing
- `listing/?`
>Gets listings belonging to specific owner

---
**Applications**

- `/applications/?`
>Gets properties belonging to single user

---
**Requests**

- `/requests/?`
>Gets all requests for a single property owner
